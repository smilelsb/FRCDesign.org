## FRCDesign.org 是什么？

1. FRCDesign.org 不只是一个资源链接合集。虽然网站包含大量资源，但几乎所有内容都是为本站专门编写，以契合我们的教学目标。

2. 一切内容都从“竞赛导向”出发。设计有竞争力的机器人能激励学生，同时培养工程能力与生活中的解决问题能力；机器人表现稳定、少出故障，赛季也会更有成就感。

3. 我们并不试图面面俱到。学习课程围绕一种 CAD 软件（Onshape）展开，并推荐一种适合大多数中等资源水平队伍的设计思路。

4. 我们尽量补上多数教程会忽略的细节。“机械臂”这个概念几乎到处都有，但如何设计一个每场比赛都可靠、不易损坏的枢轴机构？这类知识并不常见，却更加重要。

5. 设计规范很重要！我们尽力记录并传授[尽可能好的最佳实践](best-practices/index.md "最佳实践页面")，帮助你在赛季中更快完成设计。

## FRCDesign.org 由谁编写？

FRCDesign.org 凝聚了来自世界各地多支 FIRST 队伍的学生、校友与导师们数百小时的协作成果。
