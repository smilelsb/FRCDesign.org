## 什么是 CAD？

CAD（计算机辅助设计）是 “computer-aided design” 的缩写。它是一种软件工具，工程师、制造人员、建筑师以及工业领域中的许多设计人员都会使用它，把自己的想法变成可实现的设计。

CAD 会通过各种步骤和工具创建 3D 模型，帮助完成产品原型、制造和展示。文件可以发送到加工设备上，制作出对应零部件。也可以制作并打印工程图，辅助人工加工和装配。本指南和网站使用的 CAD 软件是 Onshape。

<Aside type="note" title="Software Choice（软件选择）">
本学习课程选择 Onshape，是因为它有丰富的社区资源、协作能力和工作流程。然而，无论你使用哪种软件，课程中的许多设计流程、机构基础知识以及设计手册内容仍然适用。如果你在学习本指南时选择其他软件，请注意：CAD 工作流程可能不同，而且你能获得的支持会明显更少。
</Aside>

## 为什么 FRC 要使用 CAD？

CAD 是想法与实际制造成品之间的桥梁。它之所以能带来很大改变，主要有以下几个原因：

1. **设计迭代**：队伍可以在制造前快速测试和修改设计
2. **团队协作**：多名队员可以清楚地查看并理解设计
3. **沟通交流**：所有人都能看到最终效果，并就设计方案和方向达成一致
4. **制造规划**：CAD 可以作为数字化蓝图和物料清单，帮助安排制造
5. **教学工具**：帮助新队员理解机器人机构和装配
6. **重量管理**：让队伍在设计过程中持续跟踪机器人重量
7. **效率提升**：优秀的机器人可以更快完成设计与制造，从而为程序测试和驾驶练习留出时间

**CAD 技能在现实世界中非常常用**。无论是设计和测试简单产品，还是飞机、汽车这类大型工程项目，都会用到 CAD。学习任何一种 CAD 软件，都能让你掌握可以写进简历的实用技能，也会让你之后学习其他 CAD 软件时更轻松。
