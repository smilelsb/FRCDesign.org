## 注册账号

Onshape 为学生提供免费的教育许可证。强烈建议使用教育计划，因为它允许创建私有 Document（文档），并提供其他实用功能。

要开始注册，请前往 [Onshape for Education](https://onshape.com/education-plan) 页面并选择 “Create EDU ACCOUNT”，或点击[这个链接](https://www.onshape.com/en/education/sign-up)。按照下面的幻灯片完成注册。

<Slides>
  ![](../img/signup-1.webp)
  填写注册表单中的详细信息。

  ![](../img/signup-2.webp)
  在下一个页面中，选择你是学生，并且正在基础教育阶段就读。

  ![](../img/signup-3.webp)
  最后，填写你的学校信息。使用 Onshape 的原因可以填写 “Robotics”。
</Slides>

Onshape 接下来会审核你的信息（这可能需要一些时间），然后发送一封验证邮件来激活账号。之后你会被要求设置密码，完成后就会进入你的 dashboard（控制面板）。

### 账号设置

第一次进入 Onshape 时，它会提示你设置账号，包括进行 CAD（计算机辅助设计）时使用的默认单位和鼠标操作方式。你也可以设置头像和昵称（建议使用真实姓名）。

如果你的队伍使用 Onshape，请联系导师或设计负责人，获取 Onshape classroom/team 的访问权限。

## 可选：Onshape Educator Plan

除了个人账号设置之外，如果你的队伍正在使用 Onshape，或准备切换到 Onshape，建议由一位导师或设计负责人申请 Educator plan，并将所有成员加入一个 “classroom”。Educator plan 对 FIRST 队伍免费，可以让 Document 管理更轻松。它还会为加入 “classroom” 的所有学生提供一系列功能，例如仿真、发布管理，以及课程/作业功能。

如果你对此感兴趣，可以请设计负责人或导师阅读下面链接中的 ChiefDelphi 帖子，了解更完整的概览和队伍设置流程。

[Onshape Educator Plan: What it Means for FRC Teams](https://www.chiefdelphi.com/t/onshape-educator-plan-what-it-means-for-frc-teams/446394)
