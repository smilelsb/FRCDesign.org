## 设置 FRCDesignApp

FRCDesignApp 是 Onshape 的一个插件，这是一个大型零件库，其中包含许多对 FRC 设计很有帮助的零件。

<Aside type="tip" title="Tip（提示）">
FRCDesignApp 是帮助你把组件插入文档的插件/应用，而 FRCDesignLib 才是真正的组件集合。未来 FRCDesignApp 可能会升级并加入更多功能。
</Aside>

要安装它，请前往 Onshape 应用商店中的 [FRCDesignApp 页面](https://appstore.onshape.com/apps/Manufacturers%20Models/2ZT7X5D646R3LM3ZND7LGBTYRVM4SVH6CDDGM6I=/description)。

点击 “Subscribe（订阅）”，然后点击 “Get for Free（免费获取）”。这会自动把 FRCDesignApp 添加到你的 Onshape 账号中。

<Aside type="note" title="Note（备注）">
- 订阅后，已经打开的文档中不会立刻显示 FRCDesignLib 插入器，需要刷新重新加载页面。这只会在第一次设置时出现。
- 第一次打开它时，你可能需要清除网站数据（缓存/cookies），并授权该应用
</Aside>
