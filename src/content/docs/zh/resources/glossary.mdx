## 为什么会有这份术语表
我们知道，要弄懂 FRC 设计中出现的各种术语并不容易，因此整理了这份词汇表，方便你在学习过程中随时查阅。

## 术语

**C-C（中心距）**
: “Center to Center”  “中心到中心”的距离。

**WCP**
: [West Coast Products](https://wcproducts.com/)

**TTB**
: [The Thrifty Bot](https://www.thethriftybot.com/)

**SDS**
: [Swerve Drive Specialties](https://www.swervedrivespecialties.com/)

**Turnbuckle（松紧螺杆）**
: 一种带螺纹的零件，安装在链条链节之间，用于张紧链条，简称链条张紧器。

**COG（重心）**
: Center of Gravity（重心），也常称作 COM（Center of Mass，质心）。

**OTB（越过保险杠）**
: “Over the Bumper”，通常指一种 intake（拾取机构）形式：把物件越过保险杠上方拾取。

**UTB（从保险杠下方通过）**
: “Under the Bumper”，通常指一种 intake（拾取机构）形式：把物件从保险杠下方拾取。

**V4B（虚拟四连杆）**
: “Virtual 4 Bar”（虚拟四连杆）：第二个自由度由前一个自由度通过齿轮直接驱动，而不是独立运动或通过连杆机构驱动。

**4Bar（四连杆）**
: 它由四个刚性构件（连杆）通过旋转关节（铰链）或滑动关节首尾相连，形成一个封闭的运动链。常见为曲柄摇杆机构、双曲柄机构、双摇杆机构。

**COTS（现成商业零件）**
: “Commercial Off the Shelf”，指可从供应商处直接购买的零件。

**Dead Axle（死轴）**
: 一种轴系布置：轴固定不动，和/或不通过该轴传递系统的扭矩，也称“固定轴”。

**Live Axle（活轴）**
: 一种轴系布置：轴跟随旋转，主要扭矩通过轴本身传递，也称“转动轴”。

**Zombie Axle（惰轴）**
: 多用于 pivot（枢轴/转轴）机构：它是活轴和死轴两者的结合体，它随机构一起转动，但几乎不传递扭矩；常用于在枢轴/转轴点安装编码器。

**Tube Nut（管内星形螺母）**
: 也称 “Star Nut”（星形螺母），便于在空心管端部加工内螺纹，从而将管材用螺栓固定到其他零件上，也称为“太阳花弹片”、“梅花弹片”。

**Cantilever（悬臂）**
: 水平构件仅在一侧受支撑的结构形式，也称“悬臂梁”。

**Shear（剪切）**
: 力平行于材料平面作用时产生的效应；过大的力与应力会导致紧固件因剪切而失效。也称“剪切力”。

**Torque（扭矩）**
: 使物体产生旋转运动的力矩，也称“转矩”。

**Moment of Inertia（转动惯量）**
: 旋转时惯性大小的物理量，反映了物体抵抗旋转状态改变（角加速度）的难易程度。

**End Effector（末端执行器）**
: 安装在机械臂末端的执行机构，也称“机械手”、“夹爪”。

**Linear Actuator（直线执行器）**
: 将旋转运动转换为直线运动的装置，通常使用丝杠（lead screw）实现，也称“线性电机”、“线性马达”、“电推杆”。

**PDH（配电中心）**
: Power Distribution Hub（配电中心），也称“电源分配器”。

**SparkMAX**
: 用于 REV 电机的电机控制器。

**Main Breaker（主断路器）**
: 机器人的总电源开关。

**PWM（脉宽调制）**
: Pulse Width Modulation（脉宽调制），对于传输PWM信号的线缆也称“PWM线”。

**RIO（机器人控制器）**
: RoboRIO，机器人的主控制器，现在有RoboRIO 1.0和RoboRIO 2.0两个版本。在2027年会推出全新的System Core控制器。

**Blinkn**
: REV 的 LED灯 控制器。

**Radio（通信模块）**
: 帮助机器人与 Driver Station（操控站）通信的设备，也称“路由器”。

**Limit Switch（限位开关）**
: 一种传感器；在受到物理接触或磁场触发时，可作为 soft limit（软限位）使用。

**Throughbore Encoder（通孔式编码器）**
: 允许轴从中孔穿过，并记录位置的编码器。

**Magnetic Encoder（磁性编码器）**
: 利用磁场测量位置或运动的编码器。

**3DP（3D 打印）**
: 3D printed 或 3D printing（3D 打印）的缩写。

**Billet（坯料加工件）**
: 由整块实心材料通过加工而成的零部件。

**TPI（每英寸螺纹牙数）**
: Threads per inch（每英寸螺纹牙数），指一英寸长度内紧固件上的螺纹峰数量，用于描述英制螺纹螺距（例如 #10-32 为 32 TPI，属于细牙）。

**Bevel Gear（锥齿轮）**
: 用于改变旋转轴方向，通常为 90°。

**Gusset（加强板/角撑板）**
: 加强板，通常由铝制成，用于加强两根管材之间的连接，也称“角撑板”、“角撑”。

**Box Tube（方管）**
: 方形、矩形截面的空心金属管材，也称“矩形管”。

**Tie Rod（拉杆）**
: 机器人框架内起稳固支撑作用的刚性杆件，常见于机器人车架，也称“撑杆”、“连杆支撑杆”。

**Poly/PC（聚碳酸酯）**
: polycarbonate（聚碳酸酯）的缩写，也称“聚碳酸酯”、“PC”。

**Stripping（滑牙）**
: 螺纹或螺栓头部损坏、被磨平的过程，常见为受力过载或磨损，也称“滑丝”。

**Cross Threading（交叉螺纹）**
: 装配时紧固件没有正确对准，以错误角度强行拧入造成的损坏，也称“错牙”。

**Tapping（攻丝）**
: 使用丝锥（tap）在孔内加工内螺纹的过程。

**Brainpan（倒置电气底板）**
: 把电器元件翻转安装在底板上。

**Bellypan（底盘电气底板）**
: 安装在底盘底部、用于固定电气设备的板件。

**Hardstop（硬限位）**
: 物理限位，用于防止机构或零件运动超出某一位置，限制构件最大活动范围。

**Softstop（软限位）**
: 基于软件的限制或控制，约束机构或零件的运动范围，防止超出设定参数。

**DP（径节）**
: Diametral pitch（径节），分度圆每英寸对应的轮齿数量。

**packaging（空间布局）**
: 机器人内部所有零部件的空间整合装配布局方式。

**backlash（回程间隙）**
: 传动系统中的 “空程”：可来回晃动而未带动整个系统，也称“背隙”，“传动旷量”。

**parametric（参数化设计）**
: 依托尺寸关联逻辑建模，保留设计意图的设计方式。
